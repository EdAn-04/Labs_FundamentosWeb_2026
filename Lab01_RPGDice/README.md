# Laboratorio 01 – RPG Dice Simulator 

## Descripción
Este laboratorio consiste en la creación de una **aplicación que simula el lanzamiento de un dado de rol**, utilizando **Android Studio con Jetpack Compose**, con el objetivo de reforzar los **conceptos básicos de programación** vistos en el Módulo 1.

La aplicación genera un valor aleatorio que representa el resultado del dado y lo muestra en pantalla.

---

## Objetivos
- Practicar el uso de variables y tipos de datos
- Aplicar generación de números aleatorios
- Utilizar estructuras condicionales
- Comprender el flujo básico de ejecución de una aplicación
- Relacionar la lógica del programa con la interfaz gráfica

---

## Funcionalidad
La aplicación permite simular el lanzamiento de un dado de rol:

- Se genera un número aleatorio dentro de un rango definido
- El resultado se muestra visualmente en la pantalla
- Cada ejecución representa un nuevo lanzamiento del dado

El valor generado cambia dinámicamente según la lógica del programa.

---

## Tecnologías utilizadas
- Kotlin
- Android Studio
- Jetpack Compose
- State Management (`mutableStateOf`)
- Generación de números aleatorios
- Estructuras condicionales

---

## Estructura del proyecto
- `MainActivity.kt`  
  Contiene la lógica principal del laboratorio y la interfaz gráfica.

---

## Conclusión
Este laboratorio permite aplicar los fundamentos del Módulo 1, como variables, condicionales y lógica básica, integrándolos en una aplicación Android funcional y visual.
